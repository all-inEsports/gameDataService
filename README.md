> The Service Allows you to interact with gamedata.

### 🏠 [Homepage](https://github.com/all-inEsports/gameDataService#readme)

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```

## Endpoints

```sh
GET /v1/games?page=1&perPage=10
GET /v1/games/:id
```
